# Deployment Steps

1. Create custom VPC (10.0.0.0/16)
2. Create 6 subnets across 2 AZs
3. Attach Internet Gateway
4. Create NAT Gateway in public subnet
5. Configure Route Tables
6. Launch EC2 instances in private subnets
7. Create Application Load Balancer
8. Configure Auto Scaling Group
9. Launch Multi-AZ RDS instance
10. Configure Security Groups
11. Test end-to-end connectivity
